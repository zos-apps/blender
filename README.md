## About

3D creation suite for modeling, animation, rendering, and compositing.

## Features

- 3D modeling
- Sculpting
- Animation
- Rendering
- Video editing

## Installation

### From App Store
Open the **App Store** in zOS and search for "Blender".

### Via npm
```bash
npm install @zos-apps/blender
```

## Usage

- Use viewport to navigate 3D space
- Add objects with Shift+A
- Render with F12

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `G` | Move |
| `R` | Rotate |
| `S` | Scale |
| `Tab` | Edit mode |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
